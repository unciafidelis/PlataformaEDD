Tipos de datos simples.

+

La principal característica de los tipos de datos simples consiste es que ocupan sólo una casilla de memoria; por lo tanto, una variable simple hace referencia a un único valor a la vez. En este grupo de datos se encuentran: números enteros y reales, caracteres, booleanos, enumerados y subrangos. Cabe señalar que los dos últimos no existen en algunos lenguajes de programación.

Story

+

A story is a single feature or functionality described from the end-user's perspective.

Task

+

A task is a unit of work that needs to be completed as part of a story or epic.

Bug

+

A bug represents a defect or error in the system that needs to be fixed.

Subtask

+

A subtask is a smaller, more manageable piece of work derived from a story or task.
